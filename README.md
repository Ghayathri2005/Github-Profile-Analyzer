# 🚀 GitHub Profile Analyzer

A sleek, modern web application that allows you to instantly fetch and analyze any GitHub user's profile. Built with Next.js and styled with Tailwind CSS, this tool provides real-time developer metrics, language distributions, and repository statistics—all wrapped in a premium, dark-themed UI.

## ✨ Features

- **Instant Profile Search:** Fetch real-time data for any GitHub user via the GitHub REST API.
- **Detailed Metrics:** View followers, following, public repos, and Gists at a glance.
- **Top Repositories:** Discover a user's most popular, starred, and highly-rated projects.
- **Data Visualization:** Beautiful charts (powered by Recharts) breaking down programming language usage.
- **Modern UI/UX:** A stunning dark mode interface utilizing the highly professional *Outfit* font, subtle glassmorphism, and fully responsive design.

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** TypeScript / React 18
- **Styling:** Tailwind CSS & Lucide React Icons
- **Data Visualization:** Recharts
- **API:** GitHub REST API

## 🚀 Getting Started

First, clone the repository and install the dependencies:

```bash
npm install
```

Then, start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the analyzer in action!
